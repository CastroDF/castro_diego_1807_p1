# castro_diego_1807_p1

Parcial 1 — Desarrollo y Arquitecturas Web.

Tarjeta de noticia destacada (news card) desarrollada únicamente con HTML y CSS.
La página está centrada en pantalla y la tarjeta es responsive (mobile y desktop)
gracias a Flexbox y una media query para pantallas pequeñas.

## Estructura

- `index.html` — estructura semántica de la tarjeta de noticia.
- `styles.css` — estilos, layout con Flexbox y media query (`max-width: 600px`).

## Sitio publicado

GitHub Pages: _activar desde Settings → Pages → rama `main`._
